{{Orphan|date=February 2009}} '''OpenSocialNetwork''' is a [[social network]] of open source whose project is in [[SourceForge.net]]. Its main task is to create a social network releasing the source code. == Functions == Some functions that will take this project are: * Create a social network of open source for the whole world. * Create an [[API]] to connect to other social networks. * Give a service of social networks with '''OpenSocialNetwork'''. * Using applications of communication as [[Extensible Messaging and Presence Protocol|XMPP]] to communicate in the social network. * Using other technologies of free software or open as [[OpenID]]. == External links == * [http://opensocialnet.sourceforge.net/ OpenSocialNetwork] * {{sourceforge|opensocialnet}} [[Category:Application programming interfaces]] [[Category:Social network services]] [[Category:Web 2.0]]
