{{Orphan|date=November 2006}} '''Object-Oriented Fortran''' is an [[object-oriented]] extension of [[Fortran]], in which data items can be grouped into objects, which can be instantiated and executed in parallel. It was available for [[Sun Microsystems|Sun]], [[Iris (computer)|Iris]], [[Intel iPSC|iPSC]], and [[nCUBE]], but is no longer supported. {{FOLDOC}} [[Category:Fortran programming language family]] {{Compu-lang-stub}}
