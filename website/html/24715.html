{{Orphan|date=February 2009}} {{Cleanup|date=February 2008}} '''Mirone''' - computer graphics software to manipulate and display earth sciences data in graphical form. The code is written in [[MATLAB]] and can read several common scientific data formats such as [[netCDF]]. It also provides interface to [[Generic Mapping Tools]]. ==External links== *http://w3.ualg.pt/~jluis/mirone/index.htm [[category:Earth sciences graphics software]]
