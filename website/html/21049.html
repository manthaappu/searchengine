'''Labeled Security Protection Profile''' (LSPP) is a [[protection profile]] within the [[common criteria|Common Criteria for Information Technology Security Evaluation]], a set of security functional and assurance requirements for IT products. The LSPP requirements are derived from the ''B1'' class of the [[United States|US]] [[United States Department of Defense|Department of Defense]] security standard called [[Trusted Computer System Evaluation Criteria]] (TCSEC) which was originally published in 1985. For example, [[z/OS]] V1.7 was certified [[Evaluation Assurance Level|EAL]]4+ for LSPP in 2006. == See also == * [[Common criteria]] * [http://www.commoncriteriaportal.org/files/ppfiles/lspp.pdf LABELED SECURITY PROTECTION PROFILE] [[Category:Computer security procedures]]
