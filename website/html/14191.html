{{Infobox software | name = FreeMat | logo = [[Image:FreeMat.png|48px|FreeMat icon]] | screenshot = [[File:FreeMat screenshot Linux.png|250px|]] | caption = Screenshot of FreeMat in [[Linux]] | developer = Samit Basu | latest release version = v4.0 | latest release date = {{release date and age|2009|10|09}} | programming language = [[Assembly Language]], [[C (programming language)|C]], [[C++]], [[Fortran]] | operating system = [[Cross-platform]] ([[Linux]], [[Mac OS X]], [[Microsoft Windows|Windows]]) | genre = [[List of numerical analysis software|Technical computing]] | license = [[GNU General Public License|GPL]] | website = http://freemat.sourceforge.net/ }} '''FreeMat''' is a [[free software|free]] [[open source]] [[numerical analysis|numerical computing]] environment and [[programming language]], {{cite web |url=http://freemat.sourceforge.net/ |title=FreeMat - Home |publisher=freemat.sourceforge.net |accessdate=2009-01-25 |last= |first= }} similar to [[MATLAB]] and [[GNU Octave]]. {{cite web |url=http://berkeleyscience.com/freemat.htm |title=Berkeley Science Books - Freemat and Octave |publisher=berkeleyscience.com |accessdate=2009-01-25 |last= |first= }} In addition to supporting many [[MATLAB]] functions and some [[IDL (programming language)|IDL]] functionality, it features a codeless interface to external [[C programming language|C]], [[C++]], and [[Fortran]] code, further [[Parallel Distributed Processing|parallel distributed algorithm development]] (via [[Message Passing Interface|MPI]]), and has plotting and [[3D computer graphics|3D]] visualization capabilities. {{cite web |url=http://www.linuxlinks.com/article/20080808050417963/FreeMat.html |title=FreeMat - LinuxLinks News |publisher=www.linuxlinks.com |accessdate=2009-01-25 |last= |first= }} A major disadvantage of Freemat is poor support. There is only one web forum which is moderated and questions about Freemat are often blocked by the forum moderators, making it impossible to get support.
