{{Infobox OS | name = Microsoft Auto | logo = | screenshot = | caption = | website = http://www.microsoft.com/auto | developer = [[Microsoft]] | family = [[Windows CE]] | source_model = Closed source | released = December 4, 1998 | latest_release_version = 4.1 | latest_release_date = November 27, 2009 | latest_test_version = 6.0 | latest_test_date = October 2, 2009 | marketing_target = | language = | kernel_type = | ui = | license = Microsoft [[EULA]] | working_state = | supported_platforms = | updatemodel = | package_manager = }} '''Microsoft Auto''' (also previously known as '''Windows CE for Automotive''', '''Windows Automotive''', and '''Windows Mobile for Automotive''') is an [[embedded operating system]] based on [[Windows CE]] for use on computer systems in [[automobile]]s. The operating system is developed by [[Microsoft]] through the Microsoft Automotive Business Unit that formed in August 1995. The first automotive product built by Microsoft's Automotive Business Unit debuted on December 4, 1998 as the [[AutoPC]], and also includes [[Ford Sync]] and [[Blue&Me]]. Microsoft's Automotive Business Unit has built both the software platforms used for automotive devices as well as the devices themselves. The current focus is on the software platforms and includes two products, Microsoft Auto and Windows Automotive. ==History== The Microsoft Auto operating system was originally shipped with the [[AutoPC]] that was jointly-developed by Microsoft and [[Clarion (car audio)|Clarion]]. The system was released in December 1998, and referred to the operating system itself as "Auto PC".{{cite web |url=http://www.microsoft.com/presspass/features/1998/12-4autopc.mspx |title=Drivers Keep Hands on Wheel, Eyes on Road, as Auto PC Provides Easy Access to Information and Entertainment |date=1998-12-04 |publisher=Microsoft |accessdate=2009-05-07}} Microsoft's Auto PC platform was based on Windows CE 2.0, and had been announced in January of that year.{{cite web |url=http://www.microsoft.com/presspass/press/1998/Jan98/AutoPCpr.mspx |title=Microsoft Pioneers Interactive Speech Technology |date=1998-01-08 |publisher=Microsoft |accessdate=2009-05-07}} On October 16, 2000 Microsoft officially announced the next version of the platform.{{cite web |url=http://www.microsoft.com/presspass/press/2000/Oct00/WindowsPR.mspx |title=New Microsoft Windows CE for Automotive - Reliable, Flexible and Connected |date=2000-10-16 |publisher=Microsoft |accessdate=2009-05-07}}{{cite web |url=http://windowsitpro.com/article/articleid/19473/japanese-carmakers-sign-on-to-windows-ce.html |title=Japanese carmakers sign on to Windows CE |author=Thurrott, Paul |date=2000-09-19 |publisher=Windows IT Pro |accessdate=2009-05-07}} This version of the operating system was renamed to "Windows CE for Automotive". On October 21, 2002, Microsoft announced that the platform would be renamed to "Windows Automotive".{{cite web |url=http://www.windowsfordevices.com/news/NS8826372538.html |title=Microsoft renames embedded automotive OS |date=2002-10-21 |publisher=Windows For Devices |accessdate=2009-05-07}} The version added support for development using the [[.NET Compact Framework]].
