{{Unreferenced stub|auto=yes|date=December 2009}} '''Cardea''' is the [[codename]] for portable version of [[Windows Media DRM]] for [[Computer networking|network]] [[electronic device|devices]], whose marketing name is '''Windows Media DRM for Network Devices''' (or in short form '''WMDRM-ND''') introduced by [[Microsoft]]. It is used for streaming protected digital media across a network for immediate playback. [[Janus (DRM)]] is a similar system for [[portable]] [[electronic device|devices]], but is used for synchronization. {{DEFAULTSORT:Cardea (Drm)}} [[Category:Digital rights management]] [[Category:Microsoft Windows multimedia technology]] {{Windows-stub}}
