{{Infobox Software | name = OverDrive Media Console | logo = [[Image:OverDrive Media Console Icon.gif]] | screenshot = [[Image:OverDrive Media Console.gif]] | caption = OverDrive Media Console in [[Windows XP]] | collapsible = | author = | developer = [[OverDrive, Inc.]] | released = | latest release version = 3.2 (Windows), 1.0 (Mac) | latest release date = {{release date and age|2008|11|19}} (Mac) | latest preview version = | latest preview date = | frequently updated = | programming language = | operating system = [[Microsoft Windows]] 98 or higher, [[Mac OS]] X v10.4.9 or later | platform = | language = | status = | genre = [[Media player (application software)|Media player]] | license = [[Proprietary software]] | website = [http://overdrive.com/software/omc/ OverDrive Media Console] }} '''OverDrive Media Console''' is a bare-bones proprietary media player software developed by [[OverDrive, Inc.]] for use with its download services for library, schools, and retailers. The [[Windows Media Player]]-based software allows users to download, organize, and listen to [[audiobooks]], music, and video either borrowed from an OverDrive subscribing [[Digital_Library_Reserve|library]] or purchased from an OverDrive subscribing online retailer. The software supports only a few formats: MP3, and [[Windows Media DRM]] protected [[Windows Media Audio|audio]] & [[Windows Media Video|video]]. For [[eBook]]s, OverDrive Inc, utilizes [[Adobe Digital Editions]] ([[EPUB]] & [[PDF]]), [[Mobipocket]] and [[Microsoft Reader]].
