'''Extended Window Manager Hints''', aka '''NetWM''' or '''Net WM''',[http://blackboxwm.sourceforge.net/NetWM Net WM - Blackbox] is an [[X Window System]] standard for [[X window manager|window managers]]. It defines various interactions between window managers, utilities, and applications, all part of an entire [[desktop environment]].
