{{Orphan|date=February 2009}} {{Infobox programming language |name = Squib |year = 2005 |designer = Matt Mower |developer = Matt Mower |latest_release_version = 0.5 |latest_release_date = [[2006]] |implementations = Ruby |influenced_by = [[Ruby (programming language)]] |operating_system = [[Cross-platform]] |license = [http://www.opensource.org/licenses/mit-license.php Open Source Initiative OSI - The MIT License:Licensing] |website = [http://squib.rubyforge.org/ http://squib.rubyforge.org/] }} '''Squib''' is a free, open source, weblog publishing application available under the liberal MIT Licence. Squib is written in the [[Ruby (programming language)|Ruby]] programming language and uses the [[Ruby on Rails]] web framework. ==History== Squib was written by Matt Mower in 2005. Squib was written as a replacement for his blogging package [[Radio Userland]] and an urge to learn [[Ruby on Rails|Rails]] [http://www.mattmower.org/ About Squib]. Since then it has gone through several partial rewrites as the Rails feature set developed [http://squib.rubyforge.org/ Squib]. ==Requirements== * Ruby 1.8.4+ * RubyGems 0.8+ * Rails 1.2.3 * MySQL ==Releases== Squib 0.3.6 was released on Dec 20th 2005. There was a 0.4 branch intended to support multiple weblogs that was eventually abandoned.
