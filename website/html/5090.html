'''CLX''' is an [[X Window System]] protocol client [[library (computer science)|library]] in the [[Common Lisp|Common Lisp programming language]] (Cf. [[Xlib]]). It contains [[subroutine|function]]s for interacting with an X [[Server (computing)|server]]. These functions allow [[programmer]]s to write programs without knowing the details of the [[protocol (computing)|protocol]]. CLX is completely written in Common Lisp and does not use Xlib. ==External links== * http://www.cliki.net/CLX [[Category:Common Lisp]] [[Category:Common Lisp software]] {{programming-software-stub}}
