[[Image:ecb_at91.jpg|thumb|200px|ECB AT91 SBC computer]] '''ECB_AT91''' is a small ([[Open source hardware]]) SBC ([[Single-board computer]]) based on the Atmel AT91RM9200 ARM9 processor ({{nowrap|180 MHz}}). The ECB_AT91 supports up to 64 [[Megabyte|MB]] of SDRAM. It uses the [[GNU]]/[[Linux]] operating system and measures {{nowrap|85 mm x 77 mm}}.
