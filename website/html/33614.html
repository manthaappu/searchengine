The sftp command is a client program implementing the [[SSH file transfer protocol]].[http://www.openbsd.org/cgi-bin/man.cgi?query=sftp&sektion=1 sftp man page from the openssh project] This should not be confused with running an [[FTP_over_SSH|FTP client over an SSH connection]]. sftp is part of the [[OpenSSH]] project.
