{{unreferenced|date=June 2008}} {{Portal|Software Testing}} '''National Software Testing Laboratories''', or '''NSTL''', is an [[United States|American]] company, established in 1983, which tests computer hardware and software. The company provides certification (such as [[WHQL Testing|WHQL]] and [[Microsoft Windows Mobile]] certification), quality assurance, and benchmarking services.
