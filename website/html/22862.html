{{Notability|date=November 2008}} {{Infobox Software|name=MAVID|developer=Nicolas Bray ([[UC Berkeley]]), Lior Pachter ([[UC Berkeley]])|latest_release_version=2.0.4|operating_system=[[UNIX]], [[Linux]], [[Apple Macintosh|Mac]]|genre=Bioinformatics tool|licence=Open source|website=[http://bio.math.berkeley.edu/mavid/download/ MAVID download]}} '''MAVID''' is a [[multiple sequence alignment]] program suitable for the alignment of large numbers of DNA sequences. The sequences can be small mitochondrial genomes or large genomic regions up to megabases long. The latest version is 2.0.4. The program can be used through the [http://baboon.math.berkeley.edu/mavid/ MAVID web server] or as a standalone program which can be installed with the [http://bio.math.berkeley.edu/mavid/download/ source code]. == Input/Output == This program accepts sequences in [[FASTA format]]. The output format includes: [[FASTA format]], [[Clustal]], [[PHYLIP]]. == References == N. Bray and L. Pachter, MAVID: Constrained ancestral alignment of multiple sequences, Genome Research 14 (2004), p 693--699. == External links == * [http://baboon.math.berkeley.edu/mavid/ MAVID web server] [[Category:Bioinformatics]] [[Category:Computational phylogenetics]]
