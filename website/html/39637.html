{{Refimprove|date=April 2008}}{{lowercase}} In some [[operating systems]] (e.g., [[DOS]], [[OS/2]] and [[Microsoft Windows]]) '''vol''' is a [[command (computing)|command]] within the [[command line interpreter]]s ([[Shell (computing)|shells]]) such as [[COMMAND.COM]] and [[cmd.exe]]. It is used to display the [[volume label]] and [[serial number]] on a [[logical drive]], such as a [[Hard disk drive|hard disk]] partition or a [[floppy disk]], if they exist. ==Syntax== vol [Drive:] Arguments: *'''Drive:''' This [[command-line argument]] specifies the drive letter of the disk for which to display the volume label and serial number. Note: *On Windows, the serial number is displayed only for disks formatted with [[MS-DOS]] version 4.0 or later.[http://technet.microsoft.com/en-us/library/bb491030.aspx Microsoft TechNet Vol article] *OS/2 allows the user to specify more than one drive. The vol command displays the volume labels consecutively. ==Examples== ===OS/2=== [C:\]vol C: The volume label in drive C is OS/2.
