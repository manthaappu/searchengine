{{For|the later Microsoft anti-virus product|Microsoft Security Essentials}} {{Infobox software| |name = Windows Live OneCare |logo = [[Image:Windows Live OneCare logo.png|64px|center|The Windows Live OneCare Logo]] |screenshot = [[Image:Live onecare.png|250px|A screenshot of Windows Live OneCare 2.5.2900.03]] |caption = Windows Live OneCare 2.5 |developer = [[Microsoft]] |latest_release_version = 2.5.2900.30 |latest_release_date = {{release date|2010|3|06}} |operating_system = [[Windows XP]] (32-bit), [[Windows Vista]] (32-bit/64-bit) |status = Discontinued |genre = [[Antivirus software]]{{-}} [[Backup software]]{{-}} [[Personal firewall]] |license = [[Proprietary software|Proprietary]] [[commercial software]] |website = [http://onecare.live.com http://onecare.live.com] }} '''Windows Live OneCare''' (previously known as '''Windows OneCare Live''' and formerly [[code name]]d '''A1''') was a computer security and performance enhancement service developed by [[Microsoft]] for [[Microsoft Windows]]. A core technology of OneCare was the multi-platform RAV (Reliable Antivirus),{{cite web |url=http://news.zdnet.co.uk/security/0,1000000189,39286351,00.htm |title=Microsoft: OneCare should not have been rolled out |publisher=CBS Interactive Limited |date={{date|March 16, 2007}} |accessdate={{date|2010-01-29}} }} which Microsoft purchased from [[GeCAD Software]] Srl in 2003,{{cite web |url=http://www.microsoft.com/presspass/press/2003/Jun03/06-10GeCadPR.mspx |title=Microsoft to Acquire Antivirus Technology From GeCAD Software: Future Antivirus Solutions, Windows Platform Investments Will Provide Customers With Added Protection |work=Microsoft Presspass |publisher=Microsoft Corporation |date={{date|June 03, 2003}} |accessdate={{date|2010-01-29}} }} but subsequently discontinued.{{cite web |url=http://www.infoworld.com/t/applications/microsoft-kill-popular-linux-antivirus-product-443 |title=Microsoft to kill popular Linux antivirus product |publisher=Infoworld, Inc. |date={{date|June 11, 2003}} |accessdate={{date|2010-01-29}} }} The software has been available as an annual paid subscription, which could be used on up to three computers. On {{date|November 18, 2008}}, Microsoft announced that Windows Live OneCare would be discontinued on {{date|June 30, 2009}} and will instead be offering users a new free anti-malware suite called [[Microsoft Security Essentials]] to be available before then. However, virus definitions and support for OneCare will continue until a subscription expires.{{cite web |url=http://www.microsoft.com/Presspass/press/2008/nov08/11-18NoCostSecurityPR.mspx |title=Microsoft Announces Plans for No-Cost Consumer Security Offering: New anti-malware solution will broaden PC protection and help improve Windows experience. |work=Microsoft Presspass |publisher=Microsoft Corporation |date={{date|November 18, 2009}} |accessdate={{date|2009-10-10}} }} In the end-of-life announcement, Microsoft noted that Windows Live OneCare would not be upgraded to work with [[Windows 7]] and would also not work in Windows 7 XP Mode.{{cite web |url=http://onecare.live.com/standard/en-US/activation/oc_eol_guidance.htm |title=Windows Live™ OneCare® End of Sale Guidance Page |publisher=Microsoft Corporation |accessdate={{date|2009-11-17}} }} ==History== Microsoft designed OneCare as part of a [[cloud computing|cloud]]-based computing service named [[Windows Live]] (see [http://ideas.live.com official ideas website]). Windows Live OneCare entered a [[Development stage|beta state]] in the summer of 2005. The managed beta program was launched before the public beta, and was located on BetaPlace, Microsoft's former beta delivery system. On {{date|May 31, 2006}}, Windows Live OneCare made its official debut in retail stores in the [[United States]].{{cite news |url=http://news.com.com/Microsoft+security+product+makes+official+debut/2100-7355_3-6078249.html |title=Microsoft security product makes official debut |publisher=CNET News.com |last=Evers |first=Joris |date={{date|2006-05-31}} |accessdate={{date|2006-12-16}}}} The beta version of Windows Live OneCare 1.5 was released in early October 2006 by Microsoft. Version 1.5 was released to manufacturing on {{date|January 3, 2007}} and was made available to the public on {{date|January 30, 2007}}. On {{date|July 4, 2007}}, beta testing started for version 2.0, and the final version was released on {{date|November 16, 2007}}. Microsoft acquired Komoku on {{date|March 20, 2008}} and merged its [[computer security]] software into Windows Live OneCare.{{cite web|url=http://blogs.technet.com/antimalware/archive/2008/03/20/microsoft-acquires-komoku.aspx |title=Anti-Malware Engineering Team : Microsoft acquires Komoku |date={{date|March 20, 2008}} |accessdate={{date|2009-10-10}} }} Windows Live OneCare 2.5 (build 2.5.2900.28) final was released on {{date|July 3, 2008}}. On the same day, Microsoft also released Windows Live OneCare for Server 2.5. ==Features== Windows Live OneCare currently features an integrated [[antivirus software|anti-virus]], [[firewall (networking)|firewall]], [[backup]] and restore utility, and a tune-up utility with the integrated functionality of [[Windows Defender]] for [[malware]] protection. The future addition of a registry cleaner was considered{{cite web |url=http://windowsonecare.spaces.live.com/blog/cns!C29701F38A601141!1210.entry?_c11_blogpart_blogpart=blogview&_c=blogpart#permalink |title=Counting down to shipping OneCare |author=Yoav |work= Windows Live OneCare Team Blog |date={{date|2006-03-27}} |accessdate={{date|2006-12-16}}}} but not added because "there are not significant customer advantages to this functionality".{{cite web |url=https://connect.microsoft.com/content/content.aspx?ContentID=6085&SiteID=168 |title=OneCare Beta (2.0) Chat #1 |date={{date|2007-07-26}} |accessdate={{date|2007-08-21}}}} Version 2 has features such as multi-PC and home network management, printer sharing support, start-time optimizer, proactive fixes and recommendations, monthly reports, centralized backup, and online photo backup.{{cite web |first=Gina |url=http://windowsonecare.spaces.live.com/blog/cns!C29701F38A601141!4363.entry?_c=BlogPart |title=Download the OneCare Beta (v2.0) and let us know what you think! |work=Windows Live OneCare Team Blog |date={{date|2007-07-11}} |accessdate={{date|2007-09-29}}}} Windows Live OneCare is built for ease-of-use and is designed for home users. OneCare also attempts a very minimal interface to lessen user confusion and resource use.{{Citation needed|date=April 2009}} It adds an icon to the [[Taskbar|notification area]] that tells the user at a glance the status of the system's health by using three alert colors: green (good), yellow (fair), and red (at risk).{{cite web| title = Windows Live OneCare — Getting Started| publisher = Microsoft Corporation| url = http://onecare.live.com/standard/en-us/members/getstarted.htm|accessdate ={{date|2007-09-29}}}} ===Compatibility=== Version 1.5 of OneCare is only compatible with the 32 bit versions of Windows XP and Windows Vista.{{cite web|last=Galli |first=Peter |url=http://www.microsoft-watch.com/content/security/next_release_of_onecare_still_wontsupport_vista_x64.html |title=Security - Next Release of OneCare Won't Support Vista x64 |publisher=Microsoft Watch |date=2007-01-23 |accessdate=2010-11-09}} Version 2 of OneCare supports 64 bit compatibility to Vista. In version 2.5, Microsoft released Windows Live OneCare for Server which supports Windows Server 2008 Standard 64-bit and Windows Small Business Server 2008 Standard and Premium editions. No edition of OneCare operates in safe mode. Windows Live OneCare will not support [[Windows 7]] as its development has been discontinued and replaced by [[Microsoft Security Essentials]]. ===Activation=== Windows Live OneCare requires users to activate the product if they wish to continue using it after the free trial period (90 days). This can only be achieved by having a valid [[Windows Live ID]] so that an association can be made between the account and the subscription to OneCare which has been purchased. When the product is activated, the grey message bar at the top of the program disappears, and the subscription is fully activated for 1 year from the date of activation. Users can check the status of their subscription on the ''Billing'' page at http://billing.microsoft.com/ Windows Live OneCare does not require the operating system to be checked with [[Windows Genuine Advantage]] before, during or after installation. The program can simply be installed from a [[method stub|stub]] downloaded from the http://onecare.live.com/ website or as CD media.{{cite web |url=http://www.amazon.co.uk/dp/B000KN7DDE |title=Microsoft Windows Live OneCare 1.5 (3 User Licence) (PC) |accessdate={{date|2007-07-12}} |date={{date|2007-01-30}} |publisher=[[Amazon.com|Amazon]] }} ===Protection=== Windows Live OneCare Protection Plus is the security component in the OneCare suite. It consists of three parts: *A two-way firewall blocking both incoming and outgoing traffic, as opposed to that of [[Windows XP]] pre-service pack 2 that blocks only incoming traffic. A two-way firewall protects against trojan or other viral infections running on the local computer. *An anti-virus tool that uses regularly updated anti-virus definition files. *An anti-spyware tool that uses the [[Windows Defender]] engine as a core. In version 1.0, this required the separate installation of Windows Defender and was not integrated into the OneCare interface (although it could be managed and launched from OneCare). Version 1.5 integrated the Windows Defender engine into OneCare and no longer requires separate installation. From Windows Live OneCare 1.5 onwards also monitors Internet Explorer 7 and 8 security settings and ensures that the automatic website checking feature of the Phishing Filter is enabled. ===Performance=== [[Image:winloc tuneup.png|right|250px]] Windows Live OneCare Performance Plus is the component that performs monthly PC tune-up related tasks, such as: *Disk cleanup and defragmentation. *A full virus scan using the anti-virus component in the suite. *User notification if files are in need of backing up. *Check for Windows updates by using the [[Microsoft Update]] service. ===Backup=== Windows Live OneCare Backup and Restore is the component that aids in backing up important files. Files can be backed up to various recordable media, such as external hard disks, CDs, and DVDs. When restoring files, the entirety or a subset of them can also be restored to a networked computer, as long as it's running OneCare as well. The Backup and Restore component supports [[backup software]] features such as incremental backups and scheduling. ==Criticism== [[Image:Anti-Virus Comarative No. 19 (August 2008) - False Positives.png|thumb|Windows Live OneCare tied for the fewest [[false positive]]s]] Windows Live OneCare has been criticized from both users and competing security software companies. Microsoft's acquisition of GeCAD RAV, a core technology of OneCare, and their subsequent discontinuation of that product, deprived the GNU/Linux platform (and others) of one of its leading virus scanning tools for e-mail servers, bringing Microsoft's ultimate intentions into question. On {{date|January 26, 2006}}, Windows Live OneCare was criticized by Foundstone (a division of the competing [[McAfee]] antivirus) for the integrated firewall having default [[white list]]s which allow [[Java platform|Java]] applications and [[digital signature|digitally signed]] software to bypass user warnings, since neither of those applications carry assurances that they will not have security flaws or be written with a malicious intent.{{cite web |url=http://weblog.infoworld.com/securityadviser/archives/2006/01/microsofts_onec.html |title=Microsoft's OneCare Has Holes |last=Grimes |first=Roger |work=Security Adviser |date={{date|2006-01-29}} |accessdate={{date|2006-12-16}}}} Microsoft has since responded to the criticism, justifying their decision in that Java applications are ''"widely used by third party applications, and is a popular and trusted program among our users"'', and that ''"it is highly unusual for malware to be signed."''{{cite web |url=http://windowsonecare.spaces.live.com/blog/cns!C29701F38A601141!598.entry?_c11_blogpart_blogpart=blogview&_c=blogpart |title=Firewall and Windows OneCare – a multi-layered defense |work=Windows Live OneCare Team Blog |date={{date|2006-01-31}} |accessdate={{date|2006-12-16}}}} Windows Live OneCare has also been criticized for the lack of adherence to industry firewall standards concerning intrusion detection. Tests conducted by [[Agnitum]] (the developers of Outpost Firewall) have shown OneCare failing to detect trojans and malware which hijack applications already resident on an infected machine.{{cite web| title =OneCare Firewall: a light-weight approach to a heavy-duty problem| url =http://www.agnitum.com/news/securityinsight/issues/june2006| accessdate ={{date|2007-01-28}}}} In February 2007, the first [[Windows Vista]] anti-virus product testing by [[Virus Bulletin]] magazine (a sister company of [[Sophos]], the developers of Sophos Anti-Virus) found that Windows Live OneCare failed to detect 18.6% of viruses. Fifteen anti-virus products were tested.{{cite web |title=Virus Bulletin : Independent Malware Advice. Windows Vista — February 2007. |url=http://www.virusbtn.com/vb100/archive/2007/02 |publisher=Virus Bulletin |accessdate={{date|2007-02-07}}}}{{cite web |title=Microsoft's own antivirus fails to secure Vista |url=http://news.com.com/Microsofts+own+antivirus+fails+to+secure+Vista/2100-7355_3-6156733.html?tag=cd.top |publisher=CNET News.com |last=Thurston |first=Richard |date={{date|2007-02-06}} |accessdate={{date|2007-02-07}}}} To pass the Virus Bulletin's VB100 test, an antivirus product has to detect 100% of the viruses. [http://www.AV-Comparatives.org AV-Comparatives.org] also released results that placed Windows Live OneCare last in its testing of seventeen anti-virus products.{{cite web |title=Anti-Virus comparative February 2007 |url=http://www.av-comparatives.org/ |publisher=AV-Comparatives.org |accessdate={{date|2007-03-11}}}}{{cite web |title=Microsoft's OneCare takes last place in anti-virus evaluation |url=http://www.computerworld.com/action/article.do?command=viewArticleBasic&articleId=9012078&intsrc=hm_list |publisher=Computerworld |last=Keizer |first=Gregg |date={{date|2007-03-01}} |accessdate={{date|2007-03-11}}}} In response, Jimmy Kuo of the Microsoft Security Research and Response (MSRR) team pledged to add "truly important" ("actively being spread") malware as soon as possible, while "[test detection] numbers will get better and better" for other malware "until they are on par with the other majors in this arena." He also expressed confidence in these improvements: "Soon after, [other majors] will need to catch up to us!"{{cite web |title=Anti-Malware Engineering Team : Hello World |url=http://blogs.technet.com/antimalware/archive/2007/03/15/hello-world.aspx |publisher=Technet Blogs |last=Kuo| first=Jimmy |date={{date|2007-03-15}} |accessdate={{date|2007-03-18}}}} As of April 2008, Windows Live OneCare has passed the VB100 test under Windows Vista SP1 Business Edition.{{cite web|title=VB100 results summary|url=http://www.virusbtn.com/vb100/archive/results?display=summary|publisher=Virus Bulletin|accessdate={{date|2008-08-30}}}} As of August 2008, Windows Live OneCare placed 14th out of 16 anti-virus products in on-demand virus detection rates. On the other hand, as of May 2009, Windows Live OneCare placed 2nd in a proactive/retrospective performance test conducted by AV-Comparatives.{{cite web|title=AV-Comparatives Anti-Virus Proactive/Retrospective Test|url=http://www.av-comparatives.org/images/stories/test/ondret/avc_report22.pdf|publisher=AV-Comparatives|accessdate={{date|2009-07-08}}}} AV-Comparatives.org, the test issuer, denotes that it had "very few false alarms, which is a very good achievement." The publisher also points out that false positives can cause as much harm as genuine infections, and furthermore, anti-virus scanners prone to false alarms essentially achieve higher detection scores.{{cite web |title=Anti-Virus Comparative August 2008 |url=http://www.av-comparatives.org/seiten/ergebnisse/report19.pdf |publisher=AV-Comparatives.org |accessdate={{date|2008-10-06}}}} ==See also== {{Portal|Computer security}} *[[Windows Live]] collective brand name for a set of services and software products *[[Windows Live OneCare Family Safety]] ==References== {{Reflist|2}} ==External links== *[http://onecare.live.com/ Windows Live OneCare Official Website] *[http://windowsonecare.spaces.live.com/ Windows Live OneCare Team Blog] *[http://social.microsoft.com/Forums/en-US/category/windowsliveonecare/ Windows Live OneCare Forums] *[http://support.microsoft.com/kb/921161 Virus/Suspicious file submission], [http://support.microsoft.com/kb/921159 False positive submission] {{Microsoft Security Products}} {{Windows Live}} {{Firewall software}} {{DEFAULTSORT:Windows Live Onecare}} [[Category:Windows Live|OneCare]] [[Category:Antivirus software]] [[Category:Firewall software]] [[Category:Backup software]] [[Category:Spyware removal]] [[de:Windows Live OneCare]] [[es:Windows Live OneCare]] [[fr:Windows Live OneCare]] [[ko:윈도 라이브 원케어]] [[it:Windows Live OneCare]] [[nl:Windows Live OneCare]] [[ja:Windows Live OneCare]] [[pl:Windows Live OneCare]] [[pt:Windows Live OneCare]] [[ru:Windows Live OneCare]] [[sv:Windows Live OneCare]] [[tl:Windows Live OneCare]] [[zh:Windows Live OneCare]]
