{{lowercase}} '''script''' is a [[Unix]] [[utility program|utility]] that records a [[tty]] session. scriptreplay offers a replay function to script. script captures the session in file name '''typescript''' by default, to specify a different filename follow the script command with a space and the filename as such: script recorded_session. [[Ttyrec]] provides the same kind of functionality and offers several bindings. ==External links== *[http://www.linuxinsight.com/replaying-terminal-sessions-with-scriptreplay.html script and scriptreplay in action] [[Category:Unix]] {{unix-stub}} [[de:Script (Unix)]]
