{{Primarysources|date=December 2006}} {{for|the Italian political party|Italy of Values}} The '''Integrated Data Viewer (IDV)''' from [http://www.unidata.ucar.edu Unidata/UCAR] is a [[Java language|Java]] based software framework for analyzing and visualizing [[geoscience]] data. The IDV release includes a software library and a reference application made from that software. It uses the [[VisAD]] library and other Java-based utility packages. The IDV is developed at the Unidata Program Center (UPC), part of the [[University Corporation for Atmospheric Research]], [[Boulder, Colorado]] which is funded by the [[National Science Foundation]].
