{{Orphan|date=February 2009}} {{lowercase}} '''Tree''' is a [[Computer program|program]] available for [[Unix]] and [[Unix-like]] systems. tree is a recursive [[directory (file systems)|directory]] listing program that produces a depth-indented listing of files. With no arguments, tree lists the files in the current directory. When directory arguments are given, tree lists all the files and/or directories found in the given directories each in turn. Upon completion of listing all files/directories found, tree returns the total number of files and/or directories listed. ==Poor Man's Version== 
ls -R | grep ":$" | sed -e 's/:$//' -e 's/[^-][^\/]*\//--/g' -e 's/^/   /' -e 's/-/|/'
