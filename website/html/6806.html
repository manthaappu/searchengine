{{Infobox_Software | name = Coldstone Game Engine | logo = | screenshot = | caption = | developer = [[Beenox|Beenox Studios]], [[Ambrosia Software]] | latest_release_version = 1.0.1 | latest_release_date = August 6, 2002 | latest_preview_version = | latest_preview_date = | operating_system = [[Mac OS]] 8.1 to 10.1 | genre = [[Integrated Development Environment]] | license = }} The '''Coldstone game engine''' is a game creation suite for the [[Apple Macintosh|Macintosh]] developed by [[Beenox Studios]] and [[Ambrosia Software]].{{cite web |author=Cohen, Peter |title=Do it yourself with Ambrosia's Coldstone game engine |url=http://www.macworld.com/article/3034/2002/01/coldstone.html |date=2002-01-16 |accessdate=2008-08-24 |publisher=[[Macworld]]}} ==Functionality== Although the Coldstone game creation software itself only runs on the Mac OS platform{{cite web |url=http://www.ambrosiasw.com/support/faqs/products/coldstone/Will-Coldstone-ever-be-ported-to-Windows |title=Will Coldstone ever be ported to Windows? | Ambrosia Software, Inc. |date=2005-09-15 |accessdate=2010-03-29 |publisher=[[Ambrosia Software]]}}{{cite web |url=http://www.ambrosiasw.com/support/faqs/products/coldstone/Is-Coldstone-compatible-with-OS-X-104---Tiger |title=Is Coldstone compatible with OS X 10.4 - Tiger? | Ambrosia Software, Inc. |date=2005-09-15 |accessdate=2010-03-29 |publisher=[[Ambrosia Software]]}}, it is capable of [[compiling]] stand-alone games that run on the [[Mac OS 9]] platform, the [[Mac OS X]] platform, or the [[Microsoft Windows|Windows]] platform. The software is designed to provide a full solution for creating [[Role-playing game|role-playing]] or [[Adventure game|adventure]]-style games.{{cite web |author=Cohen, Peter |title=Game Room |url=http://www.macworld.com/article/1518/2002/07/gameroom.html |date=2002-07-01 |accessdate=2008-08-24 |publisher=[[Macworld]]}} The full version of Coldstone shipped with a CD of [[royalty free]] artwork for use in one's custom games. A commercial game, titled ''[[Pillars of Garendall]]'', was produced in-house at Ambrosia to demonstrate the Coldstone software's abilities.{{cite web |author=Largent, Andy |title=Coldstone Update; PoG Review, Expansion |url=http://www.insidemacgames.com/news/story.php?ID=4288 |date=2001-11-19 |accessdate=2008-08-24 |publisher=Inside Mac Games}} ==Availability== The Coldstone game engine is currently indicated as being "Discontinued" with regard to the latest version information on the Ambrosia Software Web site.{{cite web |url=http://www.ambrosiasw.com/support/faqs/products/ |title=Products | Ambrosia Software, Inc. |accessdate=2010-03-29 |publisher=[[Ambrosia Software]]}} According to Ambrosia Software, the Coldstone software has certain flaws to be dealt with before it is made available.{{cite web |url=http://www.ambrosiasw.com/support/faqs/products/pillars-of-garendall/Whats-this-Coldstone-I-keep-hearing-about |title=What's this Coldstone I keep hearing about? | Ambrosia Software, Inc. |date=2005-09-20 |accessdate=2010-10-31 |publisher=[[Ambrosia Software]]}} ==References== {{Reflist}} ==External links== *[http://www.ambrosiasw.com/support/faqs/products/coldstone/ FAQs for the Coldstone software at Ambrosia Software] *[http://www.ambrosiasw.com/forums/index.php?showforum=95 Coldstone Web boards at Ambrosia Software] *[http://www.idevgames.com/interviews/dee-brown-beenox-coldstone-engine Interview with Dee Brown, the developer of the ColdStone engine software] {{videogame-software-stub}} {{Ambrosia Software}} [[Category:Mac OS software]] [[Category:Mac OS X software]] [[Category:Video game creation software]] [[Category:Video game engines]] [[Category:Integrated development environments]] [[Category:Video games developed in Canada]] [[Category:Video games developed in the United States]] [[Category:Ambrosia Software]] [[ru:Coldstone Game Engine]]
