{{Distinguish|Polylogarithm}} A '''polylogarithmic''' function in ''n'' is a [[polynomial]] in the [[logarithm]] of ''n'', a_k \log^k(n) + \cdots + a_1 \log(n) + a_0. In [[computer science]], polylogarithmic functions occur as the [[Big O notation|order]] of [[space complexity|memory used]] by some [[algorithm]]s (e.g., "it has polylogarithmic order"). All polylogarithmic functions are : P_l(x) = o(x^\epsilon)\, for every exponent ε > 0 (for the meaning of this symbol, see [[small o notation]]), that is, a polylogarithmic function grows slower than any positive exponent.
