{{redirect-acronym|DoaC|''[[Diary of a Camper]]''}} '''Description Of A Career''' ('''DOAC''') is a [[semantic web|semantic]] [[vocabulary]] created by [[Ramon A. Parada]] to describe professional capabilities of a worker. It has been designed to be compatible with the [[European Union|European]] [[curriculum]] ([[Europass]]) so those can be generated from a FOAF+''DOAC'' file. It includes information about education, working experience, publications, spoken languages and other skills so it can be shared and processed by any application. ==Example== Ramon Antonio Parada Website Maintainer Systems Engineer University of a Corunha 2000-06-01 2000-06-15 es en ==See also== * [[Semantic Web]] * [[FOAF (software)]] * [[hResume]] ==External links== * [http://ramonantonio.net/doac/ DOAC: Description of a Career Project Website] * [http://ramonantonio.net/doac/0.1/ DOAC Vocabulary Specification] * [http://europass.cedefop.eu.int/ European Union Europass Curriculum] {{Semantic Web}} [[Category:Knowledge representation]] [[Category:Library and information science]] [[Category:Metadata]] [[Category:Semantic Web]] {{compu-lang-stub}} [[es:Description of a Career]] [[gl:DOAC]]
