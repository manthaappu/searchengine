'''Happy''' is a [[parser generator]] written in [[Haskell (programming language)|Haskell]]. It produces haskell-code and optionally optimized code for [[Glasgow Haskell Compiler]] and it's extensions. Happy is contained (along with [[Alex (software)|Alex]], a [[lexical analysis|lexer-generator]]) in the [[Haskell Platform]]. It is available for Unix-like systems, such as Mac OS X, Linux and BSDish systems and Windows. [[Category:Parser generators]] [[Category:Haskell software]] {{programming-software-stub}}
