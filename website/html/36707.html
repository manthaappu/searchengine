A '''TPS report''' (Testing Procedure Specification) is a document used in [[software engineering]], in particular by a Software [[Quality Assurance]] group or individual, that describes the testing procedures and the testing process. The official definition and creation is provided by the [[Institute of Electrical and Electronics Engineers]] (IEEE) as follows: :'''''[[IEEE 829]] - Test Procedure Specification''''' :''The Test Procedures are developed from both the Test Design and the Test Case Specification. The document describes how the tester will physically run the test, the physical set-up required, and the procedure steps that need to be followed.
