(formerly [[Gendai Games]]) aimed primarily at non-programmers{{cite web | url=http://blogs.discovery.com/nerdabout_austin/2009/01/create-your-own.html| title=Create Your Own Video Game With Game Salad | publisher=Discovery}} for composing games in a [[drag-and-drop]] fashion{{cite web | url=http://mashable.com/2008/09/27/gendai-games/| title=Drag and Drop Game Development with Gendai | publisher=Mashable}}, utilizing visual editors and a behavior-based logic system. It is used by consumers and [[creative professional]]s such as graphic designers, animators, and game developers{{cite news | url=http://www.guardian.co.uk/technology/gamesblog/2009/mar/11/indie-game-engine-technology-development| title=Make indie games with the minimum of technical know-how | publisher=The Guardian | location=London | first=Aleks | last=Krotoski | date=March 11, 2009 | accessdate=May 3, 2010}} for [[rapid application development|rapidly prototyping]]{{cite web | url=http://www.tuaw.com/2009/09/03/gamesalad-aiming-to-bring-their-development-system-to-the-iphone/| title=Gamesalad aiming to bring their development system to the iPhone | publisher=The Unofficial Apple Weblog}}, building and [[self publishing|self-publishing]] [[cross-platform]] games and interactive media. The application runs on [[Mac OS X]] and can produce [[iPhone]] and web browser-based content.{{cite web | url=http://www.pcworld.com/article/172675/gamesalad_engine_launches_membership_plan_iphone_games.html| title=GameSalad Engine Launches Membership Plan, iPhone Games | publisher=Macworld}} On December 21, 2009, [[Macworld Conference & Expo|Macworld Expo]] partnered with [[Gendai Games]], the developer of GameSalad, for the Macworld 2010 GameSalad Challenge to promote Mac and iPhone game creation before and during the Macworld 2010 conference.{{cite web | url=http://www.macobserver.com/tmo/article/gendai_games_announces_macworld_2010_gamesalad_challenge/|title=Gendai Games Announces Macworld 2010 GameSalad Challenge | publisher=The Mac Observer}} On November 20, 2010, GameSalad unveiled a new "Free to Make" model, making basic membership free to all users, including iOS publishing. The price of their Professional account was lowered to $499. {{cite web | url=http://www.maclife.com/article/news/gamesalad_unveils_free_make_subscription_program_offers_additional_tools_ios_game_developers|title=GameSalad Unveils "Free to Make" Subscription Program with Tools for Game Devs | publisher=Mac|Life}} == Behavior system == GameSalad provides a [[graphical user interface]] for describing the rules and the behavior of game objects, called Actors, without knowledge of programming or [[scripting language]]s.{{cite web | url=http://matthewdavid.sys-con.com/node/1177016 | title=Using Game Salad for Rapid Game Development for the iPhone | publisher=SYS-CON Media}} Behaviors are components of an actor that can either instantaneously or persistently affect the actor depending on rules that compose them. The application comes with a library of behaviors (for movement, changing attribute states, affecting collision, saving, etc.) that can be inserted into rules and other behavior groups to create new behavior. == Other major features == * '''Cross-platform publishing''' - GameSalad has one common dialog-based interface for publishing to multiple platforms such as the iPhone and the web.{{cite web | url=http://gamesalad.com/products/creator | title=GameSalad Creator Product Page | publisher=Gendai Games}} * '''Real-time editing''' -- Users can edit a scene while the game is running. The initial state of actors can be toggled to be displayed translucently to show their original orientation when the scene started playing. * '''Game preview''' -- GameSalad has a specific preview mode for debugging and testing the performance and memory usage of games on the desktop and the iPhone. A viewer application can be installed separately onto an iPhone so that users can click a toolbar button in GameSalad to upload games onto the iPhone over a wireless network. * '''Scene editor''' -- Users can place and manipulate actors in a scene. Actors are added to the scene by dragging and dropping. Actors in a scene can be organized into different [[2d computer graphics#Layers|layers]] to change how actors are visualized (rendering order, [[parallax scrolling]], etc.). * '''Dashboard''' -- With the dashboard, users can browse uploaded games, run tutorials, message other GameSalad members, and download shared templates upload by other GameSalad members. * '''Integrated physics''' -- GameSalad is utilizes a [[rigid body|rigid-body]] physics simulator for handling realistic motion and collision. Users can manage and optimize how objects collide by organizing actors with [[Tag (metadata)|tags]]. Users can choose to have an actor collide with a group of many other types of actors. * '''Expressions''' -- For advanced users, GameSalad has an expression editor to define complex behavior and state changes with [[Expression (mathematics)|mathematical expressions]] and a library of [[Function (mathematics)|functions]]. == External links == * [http://gamesalad.com GameSalad Website] Official Web site == References == {{reflist}} [[Category:Video game engines]] [[Category:Mac OS X software]] [[Category:Video game creation software]] [[ru:GameSalad]]
