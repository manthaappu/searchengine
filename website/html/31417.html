A '''real time clock alarm''' is a feature that can be used to allow a [[computer]] to 'wake up' after shut down to execute tasks every day or on a certain day. It can sometimes be found in the 'Power Management' section of a [[motherboard]]'s [[BIOS]] setup. However, newer BIOS setups do not include an RTC alarm option, although it can still be set from within user applications. [[Wake On LAN]], [[Wake-on-ring|Wake on ring]], and [[Intelligent_Platform_Management_Interface|IPMI]] functions could also be used to start a computer after it is turned off.
