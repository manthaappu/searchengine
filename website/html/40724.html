'''XOTcl''' is an [[object-oriented]] extension for the [[Tcl|Tool Command Language]] created by Gustaf Neumann and Uwe Zdun. It is an extension of the MIT [[OTcl]]. XOTcl supports [[metaclass]]es. [[Class (computer science)|Class]] and [[Method (computer science)|method]] definitions are completely dynamic.
