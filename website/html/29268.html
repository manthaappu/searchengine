It uses a set of 'conduits' for the exchange of information from different Palm applications. pilot-link provides [[command line]] tools to perform synchronisation tasks, such as the pilot-xfer command, which can install the standard Palm database ([[PDB (Palm OS)|PDB]]) files to the connected device.
