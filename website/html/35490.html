The tool detects and automatically enables source address forgery. Stacheldraht uses a number of different DoS attacks, including: [[User Datagram Protocol|UDP]] flood, [[Internet Control Message Protocol|ICMP]] flood, [[Transmission Control Protocol|TCP]] [[SYN flood]] and [[Smurf attack]].
