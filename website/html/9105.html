{{About|the Microsoft Windows software|the electronic device|DVD player}} {{Infobox Windows component | name = DVD Player | logo = Dvdplay Icon.jpg | screenshot = Dvdplay Screenshot.jpg | caption = | included_with = [[Microsoft Windows]] [[Windows 98|98]], [[Windows Me|Me]], [[Windows 2000|2000]] | replaces = | replaced_by = [[Windows Media Player]] | related_components = [[CD Player (Windows)|CD Player]] | support_status = Discontinued as of [[Windows XP]] }} '''DVD Player''' (also known as dvdplay.exe) was a [[computer program]] included in some versions of [[Microsoft Windows]] that [[autoplay]]s a [[DVD]] disc if a supported [[MPEG-2]] [[codec|decoder]] was present. DVD Player was introduced in [[Windows 98]][http://www.microsoft.com/technet/archive/win98/reskit/part2/wrkc12.mspx?mfr=true Windows 98 Multimedia]. When the DVD Player is launched, it searches all local drives in alphabetical order, starting with the C drive, looking for a Video_TS folder. When this folder is located, the data file within it is loaded, and video streaming begins. If this folder exists on a drive that comes before the DVD drive, the player will try to play the data in the first folder it finds. In Windows 98/Me and Windows 2000, DVDPlay.exe only plays DVDs if a hardware-based decoder is present. In [[Windows Me]], DVDPlay supports software-based decoders.[http://support.microsoft.com/kb/272985 Description of DVD Player in Windows Millennium Edition]. With Windows XP, DVDPlay.exe was dropped as a separate application in favor of DVD functionality introduced into [[Windows Media Player]].
