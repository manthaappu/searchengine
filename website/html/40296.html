{{Distinguish|Windows Marketplace|Windows Phone Marketplace}} {{Infobox Software |name = Windows Marketplace for Mobile |logo = [[File:Windows Marketplace for Mobile icon.png]] |screenshot = |caption = |author = |developer = [[Microsoft]] |released = |latest release version = |latest release date = |latest preview version = |latest preview date = |frequently updated = |programming language = |operating system = [[Windows Mobile]] 6.x |platform = |size = |language = |status = |genre = [[Digital distribution]] |license = [[Proprietary software|Proprietary]] |website = [http://marketplace.windowsphone.com/ Marketplace for Windows Mobile] }} '''Windows Marketplace for Mobile''' is a service by [[Microsoft]] for its [[Windows Mobile]] platform that allows users to browse and download applications that have been developed by third-parties. The application are available for use directly on Windows Mobile 6.5 devices, and on personal computers. It was announced at the 2009 [[Mobile World Congress]], and began operation on 6 October 2009, featuring an initial 246 applications.[http://www.pcworld.com/article/173225/whats_inside_the_windows_mobile_marketplace.html] Initially it is available only for Windows Mobile 6.5, and was made available to all 6.x versions few weeks later.http://windowsteamblog.com/blogs/windowsphone/archive/2009/11/16/windows-marketplace-for-windows-mobile-6-0-and-6-1.aspx On February 15, 2010 Microsoft announced its next generation mobile platform named [[Windows Phone 7]]. With this newly redesigned platform, Microsoft will use a separate app store, called [[Windows Phone Marketplace]], for its new mobile OS, and will not share the same apps, UI, or some features of the current version for Windows Mobile 6.x because of the many differences in user interface and experience between these two platforms. ==Pricing and features== Windows Marketplace for Mobile, available through applications for Windows Mobile and personal computers, offers a 24-hour return policy for buyers, and 70% of each application sale will be paid to developers.{{cite web |url=http://i.gizmodo.com/5199933/giz-explains-all-the-smartphone-mobile-app-stores |title=Giz Explains: All The Smartphone Mobile App Stores |author=Herrman, John |date=2009-04-06 |publisher=Gizmodo |accessdate=2009-04-06}} A one-time $99 USD fee for developers to list an unlimited number of paid applications or up to five free applications yearly in the store is charged by Microsoft. After five free applications have been listed for the year, each additional free application can be listed for a $19.99 USD fee. Microsoft accepts payment from buyers through credit cards and [[Mobile network operator|carrier]] charges. Microsoft announced that students that are members of Microsoft's [[DreamSpark]] program can submit applications to the store for free, waiving the $99 USD fee. Carriers also have the ability to create a [[brand]]ed section of the store to market carrier-specific applications and services for their customers. ==Content restrictions== Microsoft enforces certain rules before developers can submit apps to the Marketplace.http://developer.windowsphone.com/resources/en-us/MarketplaceContentPolicies.pdf Some of these restrictions include: * Sex / Nudity – Images that are suggestive or sexual (e.g. sexually provocative touching, bondage, masturbation) or provocative images that reveal nipples, genitals, buttocks, or pubic hair. * Any adult and/or borderline adult content (images or text). * Content that generally falls under the category of pornography. * Content that depicts or suggests prostitution. * Any content depicting sexual fetishes. * Any content of a sexual nature depicting children or animals. ==History== Rumors about the service began to surface around the internet in September 2008, whereas it was referred to as "Skymarket", and was to appear with the Windows Mobile 7 release.{{cite web |url=http://arstechnica.com/microsoft/news/2009/02/ballmer-windows-mobile-65-phones-are-just-windows-phones.ars |title=Windows Mobile 6.5 UI a big improvement;
