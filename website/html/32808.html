'''The Special Interest Group "Model-Driven Software Engineering" (SIG-MDSE)''' organizes the MDSE as an [[academic conference]]. This special interest group focus on model driven (or model based) and architecture centered software engineering techniques and tools, e.g. OMG's [[MDA]] (Model Driven Architecture). It is a platform for all interested persons to exchange experiences in the context of workshops, conferences etc. The activities are mainly in Germany.
