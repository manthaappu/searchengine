{{Refimprove|date=April 2008}}{{lowercase}} In [[computing]], '''runas''' is a [[command (computing)|command]] in the [[Microsoft Windows]] line of [[operating system]]s that allows a user to run specific tools and programs with different [[Privilege (computing)|permissions]] from those the user's current [[Logging (computer security)|logon]] provides. It is similar to the [[Unix]] command [[sudo]], but neither runas nor [[User Account Control|UAC]] is sudo; rather, they fully [[sudo#Runas.2C_su_and_sudo|impersonate]] another user rather than just adding privileges. The runas command was introduced with the [[Windows 2000]] operating system and is also included in the [[Inferno (operating system)|Inferno]] operating system. ==Syntax== ===Microsoft Windows=== runas [{/profile | /noprofile}] [/env] [/netonly] [/smartcard] [/showtrustlevels] [/trustlevel] /user: program ===Inferno=== runas writes the user to /dev/user and invokes cmd with the given [[Command-line argument|arguments]]. runas user cmd [arg...] Note: The command is only invoked if setting of the user name succeeds. ==Examples== ===Microsoft Windows=== runas /noprofile /user:machine\administrator cmd runas /profile /env /user:domain\admin "mmc %windir%\system32\dsa.msc" runas /user:user@domain.example.org "notepad C:\filename.txt" ==See also== *[[Comparison of privilege authorization features]] *[[Principle of least privilege]] *[[User Account Control]] ==References== {{reflist}} ==External links== *[http://technet.microsoft.com/en-us/library/bb490994.aspx Microsoft TechNet Runas article] *[http://technet.microsoft.com/en-us/sysinternals/cc300361.aspx Sysinternals ShellRunas] {{Windows commands}} [[Category:Operating system security]] [[Category:Microcomputer software]] [[Category:Computer security software]] [[Category:Windows administration]] {{software-stub}} {{windows-stub}} [[de:Runas]]
