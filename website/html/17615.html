{{about||the aroma compound known as IPMP found in wine|methoxypyrazines|the redundant IO technology|Multipath I/O|the Intellectual Property Management and Protection|MPEG-4}} '''IPMP''', or '''IP network multipathing''', is a facility provided by [[Solaris (operating system)|Solaris]] to provide fault-tolerance and load spreading for [[network interface card]]s (NICs). With IPMP, two or more NICs are dedicated for each network to which the host connects. Each interface is assigned a static "test" [[IP address]], which is used to assess the operational state of the interface. Each virtual IP address is assigned to an interface, though there may be more interfaces than virtual IP addresses, some of the interfaces being purely for standby purposes. When the failure of an interface is detected its virtual IP addresses are swapped to an operational interface in the group. The IPMP load spreading feature increases the machine's bandwidth by spreading the outbound load between all the cards in the same IPMP group.
