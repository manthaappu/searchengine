'''ODRL''' (''Open Digital Rights Language'') is an [[XML]]-based standard [[Rights Expression Language]] (REL) used in [[Digital Rights Management]] systems and open [[content management system]]s. ODRL is managed by an open organization that's open to public participation. It has created a profile that supports [[Creative Commons]] licenses and is working on a profile for geospatial data and a profile for [[Dublin Core Metadata Initiative]] (DCMI) metadata. The ODRL Version 2.0 is currently working on major revisions of the new version of ODRL.
