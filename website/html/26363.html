'''Next Byte Codes''' (NBC) is a simple language with an [[assembly language]] syntax that can be used to program [[Lego Mindstorms NXT]] programmable bricks. The command line compiler outputs NXT compatible machine code, and is supported on Windows, Mac OS and Linux.[http://www.pcwelt.de/news/software/77325/index.html Update für NBC: Programmiersprache für Lego Mindstorms NXT] It is maintained by John Hansen, a Mindstorms Developer Program member.[http://mindstorms.lego.com/MeetMDP/JohnHansen.aspx Mindstorms Developer Program: John Hansen] The NBC compiler is available under the [[Mozilla Public License]]. The [[Integrated development environment|IDE]] is [[Bricx Command Center]]. The NBC [[debugger]] was developed by SorosyDotCom and is available for download as freeware.
