'''TOTP - Time-based One-time Password Algorithm ''' is an extension of the [[HMAC|HMAC-based]] [[One-time password|One Time Password]] algorithm [[HOTP]] to support time based moving factor. TOTP was submitted to the IETF as an Internet-Draft. it is a cornerstone of [[Initiative For Open Authentication]] (OATH). ==Applications== TOTP can be used to authenticate a user in a system via an authentication server. Also, if some more steps are carried out, the user can also authenticate the validation server. ==History== The TOTP draft was developed through the collaboration of several OATH members in order to create an industry-backed standard. It complements the event-based one-time standard HOTP, and offers end user organizations and enterprises more choice in selecting technologies that best fit their application requirements and security guidelines. In addition, OATH also submitted the final version of the OCRA (OATH Challenge-Response Algorithms) specification to the IETF.
