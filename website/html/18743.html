{{Infobox software | name = IntelliJ IDEA | logo = [[Image:IntelliJ IDEA logo.gif‎|200px]] | screenshot = [[Image:IntelliJIDEA 9 Community.png|250px]] | caption = Demonstration of IntelliJ IDEA | collapsible = yes | developer = [[JetBrains]] | latest release version = 10.0.1 | latest release date = {{release date|2010|12|23}} | latest preview version = | latest preview date = | operating system = [[Cross-platform]] | programming language = [[Java (programming language)|Java]] | genre = [[Integrated development environment|Java IDE]] | license = [[Apache License|Apache 2.0]] for Community Edition (starting with version 9.0), [[Proprietary software|Proprietary]] for Ultimate Edition | website = http://www.jetbrains.com/idea/ }} '''IntelliJ IDEA''' is a commercial [[Java (programming language)|Java]] [[Integrated Development Environment|IDE]] by [[JetBrains]]. It is often simply referred to as 'IDEA' or 'IntelliJ'. A 30-day fully functional trial of the [http://www.jetbrains.com/idea/ IntelliJ IDEA] commercial edition for various platforms can be [http://www.jetbrains.com/idea/download/index.html freely downloaded]. Also available is an [[open source]] Community Edition.{{cite web|url=http://blogs.jetbrains.com/idea/2009/10/intellij-idea-open-sourced/|title=IntelliJ IDEA Open Sourced|accessdate=October 15, 2009|date=October 15, 2009}} ===The philosophy=== The first version of IntelliJ IDEA appeared in January 2001, and at the time was the only available Java IDE with advanced code navigation and [[code refactoring]] capabilities integrated.[http://www.jetbrains.com/idea/features/refactoring.html Multi-language Refactorings] ===Most notable current features=== The latest version of IDEA (9.0) includes [[Unified Modeling Language|UML]]-like class diagrams, visual Hibernate modeling, Spring 3.0 support, Dependency and [[Data-flow analysis|Data Flow Analysis]], [[Apache Maven]] support as well as a lot of other features and technologies. A complete list can be found at [http://jetbrains.com/idea/features/index.html the features page].
