'''Microsoft DirectCompute''' is an [[application programming interface]] (API) that supports [[GPGPU|general-purpose computing on graphics processing units]] on [[Microsoft]] [[Windows Vista]] and [[Windows 7]]. DirectCompute is part of the Microsoft [[DirectX]] collection of APIs and was initially released with the DirectX 11 API but runs on both DirectX 10 and DirectX 11 graphics processing units.[http://www.nvidia.com/object/directcompute.html DirectCompute Support on NVIDIA's CUDA Architecture GPUs][http://blogs.amd.com/nigeldessau/2009/09/22/speaking-directly/ Nigel Dessau, senior VP and CMO at AMD, about DirectCompute] The DirectCompute architecture shares a range of computational interfaces with its competitors - the [[Khronos Group]]'s [[OpenCL|Open Computing Language]] (OpenCL) and [[NVIDIA]]'s [[CUDA|Compute Unified Device Architecture]] (CUDA). ==See also== *[[OpenCL]] *[[CUDA]] *[[GPGPU]] ==References== ==External links== *[http://www.microsoft.com/video/en/us/details/6ef116dc-b1d9-41db-8a7b-db1932ff72a5 Example of DirectCompute for Next Generation Game Effects] *[http://blogs.amd.com/play/2009/09/09/directx-11-%E2%80%93-what-to-expect/ DirectCompute for AMD] *[http://www.nvidia.com/object/cuda_directcompute.html Cuda DirectCompute for NVIDIA] *[http://benchmarkreviews.com/index.php?option=com_content&task=view&id=8374&Itemid=47 AMD and CyberLink announcement about Accelerated Video Applications with Windows 7 DirectX 11 DirectCompute] *[http://www.ngohq.com/news/16710-first-directcompute-benchmark-released.html DirectCompute Benchmark] *[http://www.opengpu.org/ DX11 info from Open Source GPU site]{{zh-cn}} {{DEFAULTSORT:Directcompute}} [[Category:GPGPU]] [[ko:다이렉트컴퓨트]] [[ja:DirectCompute]] [[ru:DirectCompute]] [[zh:DirectCompute]]
