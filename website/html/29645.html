{{Orphan|date=February 2009}} '''Portable Forth Environment''' (or '''PFE''') is an [[Open source]] implementation of the [[Forth (programming language)|Forth programming language]]. It conforms to the ANSI standard for Forth (ANS Forth). Its first release was in 1993, by Dirk-Uwe Zoller. == External links == * [http://pfe.sourceforge.net/ PFE] official home page {{software-stub}} [[Category:Forth programming language family]]
