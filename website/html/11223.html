DKMS enables [[Linux kernel|kernel]] [[device driver]]s to be automatically rebuilt when a new kernel is installed. DKMS can be used in both directions: To automatically [[recompile]] all modules if a new kernel version is installed, or to install new module (driver) versions on an existing system without any need for manual compilation or precompiled packages. This for example makes it possible to use new [[graphics card]]s on an older Linux system. DKMS was written by the Linux Engineering Team at [[Dell]] in 2003.
