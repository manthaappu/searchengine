{{Infobox software | name = Rawstudio | logo = | screenshot = [[File:RawstudioCA.png|200px]] | caption = Rawstudio, v. 1.0 | author = Anders Brander & Anders Kvist | developer = | released = | latest release version = 1.2 | latest release date = {{release date|2009|04|09}} | latest preview version = | latest preview date = | operating system = Linux, Mac | platform = | language = | genre = [[Graphical user interface|GUI]] for [[Raw image format|RAW]] decoding software | license = [[GNU General Public License]] [[GPLv2|version 2]] | website = [http://rawstudio.org rawstudio.org] }} '''Rawstudio''' is a stand-alone application, using the [[GTK+]] UI toolkit, to read and manipulate images in [[Raw image format]]s from digital cameras. It is designed for working rapidly with a large volume of images, unlike similar tools, such as [[UFRaw]], which are designed to work on one image at a time. Rawstudio is currently at a very early stage of development and currently only the latest revision supports [[demosaicing]]. Rawstudio reads RAW images using [[DCRaw]] and supports [[color management]] using [[LittleCMS]] to allow the user to apply color profiles (see also [[Linux color management]]).
