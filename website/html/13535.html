There are several different technologies that use the term '''file shadowing''', but the intent of file shadowing within these technologies is to provide an exact copy (or [[Mirror (computing)|mirror]] of a set of data). For file shadowing to be effective, the shadow needs to exist in a separate physical location than the original data.
