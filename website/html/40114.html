{{Multiple issues|notability=August 2009|refimprove=August 2009}} A '''WAD''' file is a file archive containing .APP files, which can be extracted and installed on a [[Wii]] [[Video game console|gaming console]]. They often contain information for the Wii, such as System Menu, IOS, Channels etc. Although they are mainly used to install custom channels and IOS, they have also been linked to piracy, as users can install pirated [[WiiWare]] and [[Virtual Console]] titles through the use of WADs. WADs are generally installed using the homebrew application WAD Manager.http://wiibrew.org/wiki/WAD ==References== {{Reflist}} {{DEFAULTSORT:Wii Wad}} [[Category:Video game consoles]] [[Category:Wii]]
