{{About|software development tools|other uses|Raise (disambiguation)}} '''RAISE''' (''Rigorous Approach to Industrial Software Engineering'') was developed as part of the European [[ESPRIT]] II LaCoS project in the 1990s, led by [[Dines Bjørner]]. It consists of a set of tools based around a [[specification language]] (RSL) for [[software development]].
