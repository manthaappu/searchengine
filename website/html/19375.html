An '''irregular matrix''', or '''ragged matrix''', can be described as a [[matrix (mathematics)|matrix]] that has a different number of elements in each row. Ragged matrices are not used in [[linear algebra]], since standard matrix transformations cannot be performed on them, but they are useful as [[array data structure|array]]s in [[computing]]. Irregular matrices are typically stored using [[Iliffe vector]]s.
