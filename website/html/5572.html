Cascading is based on the concatenation of several classifiers, using all information collected from the output from a given classifier as additional information for the next classifier in the cascade. Unlike voting or stacking ensembles, which are multiexpert systems, cascading is a multistage one. == See also == {| || * [[Boosting]] * [[Bootstrap aggregating]] |} == References == ===Notations=== * J.
