{{Lowercase|ltrace}} '''ltrace''' is a debugging utility in [[Linux]] to monitor the [[library call]]s used by a program and all the signals it receives. It can also show system calls, used by a program. == See also == * [[strace]] - system call tracer for linux * [[ktrace]] - system call tracer for *BSD * [[Truss (Unix)|truss]] - classic system call tracer * [[dtrace]] - Solaris / OS X / BSD kernel tracing tool. * [[SystemTap]] - Linux kernel tracing tool. * [[latrace]] - [http://latrace.sourceforge.net/latrace.html dynamic library call tracer] using a LD_AUDIT libc feature ==External links== * [http://www.ltrace.org/ Home page] * [http://linux.die.net/man/1/ltrace ltrace man page] * Rodrigo Rubira Branco, [http://www.kernel.org/doc/ols/2007/ols2007v1-pages-41-52.pdf ''Ltrace Internals''], [http://www.linuxsymposium.org/ Ottawa Linux Symposium] 2007 [[Category:Unix programming tools]] {{unix-stub}} [[pl:ltrace]]
