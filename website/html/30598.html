'''Pydoc''' is a documentation module for the [[programming language]] [[Python (programming language)|Python]]. Similar to the functionality of [[Perldoc]] within [[Perl]], Pydoc allows Python programmers to access Python's documentation help files, generate [[HTML]] pages with documentation specifics, and find the appropriate module for a particular job.[http://epydoc.sourceforge.net/stdlib/pydoc-module.html Pydoc. Accessed 11/27/2010] Pydoc can be accessed from a module-specific [[GUI]],[http://onlamp.com/pub/a/python/2001/04/18/pydoc.html Using Pydoc. Accessed 11/27/2010] from within the Python interpreter, or from a [[command line]] shell.[http://www.ibm.com/developerworks/linux/library/l-cpmod.htm Charming Python: pydoc and distutils modules. Accessed 11/27/2010] Developed by Ka-Ping Ye, it is included by default in all versions of Python since Python 2.1 and is available for download for 1.5.2, 1.6, and 2.0. == References == {{Reflist}} [[Category:Python programming language]]
