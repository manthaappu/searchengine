{{Notability|date=November 2009}} {{Infobox software | name = Caddie | logo = [[Image:caddie (CAD software) logo.jpg|184px]] | screenshot = | caption = | author = | developer = Advanced Computer Solutions (Europe) Limited | released = 1986 | latest release version = 16 | latest release date = September 2010 | operating system = [[Microsoft Windows|Windows]] | platform = | language = | status = | genre = [[Computer-aided design|CAD]] | license = [[Proprietary software|Proprietary]] | website = [http://www.caddiesoftware.com/ www.caddiesoftware.com] }} '''Caddie''' is a mid-range [[Computer-aided design|computer-assisted draughting]] (CAD) software package for [[2D computer graphics|2D]] and [[3D computer graphics|3D]] design. It is used primarily by [[architect]]s, but has tools for [[surveying|surveyor]]s and [[mechanical engineering|mechanical]], [[civil engineering|civil]] and [[construction engineering|construction]] engineers. It was initially designed as a electronic [[drawing board]], using concepts and tools clearly related to a physical board.http://www.revitup.co.za/main/page_10862.php A competitor of Caddie uses "electronic drawing board" as a slur. Caddie requires a [[Universal Serial Bus|USB]] [[dongle]].http://www.xt.co.za/Caddie_Overview.htm Without the dongle, the program can be used as a viewer and plot station for any [[.dwg|DWG]] drawings, but it can't save drawings after the 14 day evaluation has expired. Caddie works on [[Windows XP]], [[Windows Vista]] and [[Windows 7]].http://caddiesoftware.com/caddiesoftware/hardware-requirements.htmlhttp://www.microsoft.com/windows/compatibility/Windows-7/en-us/Details.aspx?type=Software&p=Caddie%20Professional&v=Advanced%20Computer%20Solutions&uid=15&l=en&pf=3&pi=5&c=Graphics%20%26%20Printing&sc=Image%20Editing&os=32-bit == Version history == Caddie started life as student project called Michael Angelo in 1985.http://www.caddiesoftware.com/pdf/caddie14webinfo.pdf The first release version was called Caddie and fit on a single 360 k [[floppy disk]], and was designed for the [[IBM Personal Computer XT]]. Caddie was one of the first CAD tools that utilised microcomputers and did not require a [[mainframe computer]] with [[workstation]] access. Version 1 of Caddie was released in 1986, for [[MS-DOS]]. The first version for [[Microsoft Windows]] ([[16-bit]]) was released in September 1993. The [[32-bit]] Windows version was released in May 1997.http://web.archive.org/web/19981203075505/www.caddie.co.za/history.html Version 6 was available by 2000. In the same year, a lite version of Caddie was available, called Caddie Budget,http://web.archive.org/web/20000303154429/www.caddie.co.uk/caddie.htm with the non-lite version being called Caddie Professional. Version 7 was available in 2001.http://web.archive.org/web/20010630120314/www.caddie.co.uk/budget.htm Caddy Budget Architectural 7 retailed in the UK for Â£595 in 2003.http://web.archive.org/web/20030324170034/www.caddie.co.uk/c7wbudget.pdf Version 9 was released in January 2003.http://web.archive.org/web/20030207141222/http://www.caddie.co.uk/ Caddie initially used its own proprietary file format CEX (Caddie Exchange format), but since version 10, Caddie's native format is the open format [[OpenDWG]].http://www.engineeringtalk.com/news/oen/oen105.htmlhttp://www.microstationconnections.com/print_article.php?cpfeatureid=205 Version 10 was released in March 2005.http://web.archive.org/web/20050404231918/www.caddiesoftware.com/ Version 11 was released in January 2007 and was the first version compatible with Windows Vista.http://web.archive.org/web/20070927093747/www.caddiesoftware.com/ Caddie 12 was released in September 2008.http://web.archive.org/web/20080430133103/www.caddiesoftware.com/ The latest version of Caddie is version 16, released in September 2010. There are four versions of the product, namely Caddie Professional (the full version), Caddie Budget (the lite version), Caddie Vio (for photo-realistic [[rendering (computer graphics)|rendering]], based on Lightworkshttp://www.lightworkdesign.com/news_archive/2008) and Caddie Educational, which is licensed to students even though it has all the functionality of Caddie Professional.http://caddiesoftware.com/caddiesoftware/products.html == Company history == Caddie was originally developed in [[South Africa]] in 1985, and it quickly established a significant presence in its home market. The product's position in South Africa began to erode in the 1990s following the lifting of [[Disinvestment from South Africa|economic sanctions against South Africa]], as other CAD packages entered that market. A Microsoft Windows version of Caddie was not released until 1993. In 1998, Caddie was acquired by the South African software development company Billcad Holdings. By this time, Caddie had about 11 000 users.http://web.archive.org/web/19981206052916/www.caddie.co.za/default1.htm In 1999, Billcad became Planit Technology.http://www.moneyweb.co.za/mw/view/mw/en/page305985?oid=45933&sn=Daily%20news%20detail In early 2001, Planit Technologies went into [[liquidation]].http://www.greengazette.co.za/pages/2007/09/Legal%20Notices/20070914%20-%20Legal%20Notice%20No%2030277%20of%2014-Sep-2007,%20Volume%20507,%20A,%20102.pdf By March 2002, the British company Advanced Computer Solutions (ACS) bought all the rights from the South African holders and also acquired the Caddie operation within South Africa.http://web.archive.org/web/20030207141222/http://www.caddie.co.uk/ ACS was previously Caddie's British [[distribution (business)|distributor]].http://web.archive.org/web/20000303105143/http://www.caddie.co.uk/ Caddie was distributed by Advanced Computer Solutions (Europe) Limited ACS has offices in South Africa and the United Kingdom.http://caddiesoftware.com/caddiesoftware/company/office-locations.html ==External links== * [http://www.caddiesoftware.com Caddie Software website] * [http://web.archive.org/web/19981203075505/www.caddie.co.za/history.html Early history of Caddie] * [http://www.interbuild.com/ExhibitorLibrary/283/Caddie_Testimonial_Allister_Godfrey_2.pdf An Architect's view of using Caddie] * [http://www.architect-net.co.uk Caddie UK User Group] ==References== {{reflist}} {{DEFAULTSORT:Caddie (Cad System)}} [[Category:Computer-aided design software]]
