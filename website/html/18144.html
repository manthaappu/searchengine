The '''Indexed Database API''', or '''Indexed DB''' (formerly WebSimpleDB) is a proposedhttp://www.w3.org/TR/IndexedDB/ web browser standard interface for a local database of records holding simple values and hierarchical objects. Indexed DB was initially proposed by [[Oracle Corporation|Oracle]] in 2009.http://www.w3.org/TR/2009/WD-WebSimpleDB-20090929/ Indexed DB could be used for browser implemented functions, such as bookmarks, as well as web applications, such as email.
