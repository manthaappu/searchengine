{{unreferenced|date=May 2010}} {{notability|products|date=April 2009}} {{Infobox_Software | name = WYMeditor | latest_release_version = 0.5-rc3 | latest_release_date = {{release date|2010|06|3}} | operating_system = [[Cross-platform]] | genre = [[HTML editor]] | license = [[GNU General Public License]], [[MIT License]] | website = [http://www.wymeditor.org/ www.wymeditor.org] }} '''WYMeditor''' is an [[open source]] [[WYSIWYM]] [[text editor]] written in the [[JavaScript]] programming language for editing content on web pages. It is based on the [[jQuery]] [[JavaScript library|JavaScript framework]]. It differs from other embeddable text editors such as [[FCKeditor]] and [[TinyMCE]] in that it concentrates on the semantics and meaning of content leaving out visual details. Unlike [[WYSIWYG]] editors, it explicitly shows the [[XHTML]] structure of content to the user. Presentation and visual coherence is added using [[CSS]] which is either provided prepackaged, or can be customized. WYMeditor is web server agnostic meaning it can be integrated into web pages built with any server side language.
