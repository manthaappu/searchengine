'''Communote''' is a [[microblogging]] tool and service launched in December 2008.{{cite web|accessdate=2009-06-17|url=http://www.communote.com/blog/2008/12/07/communote-launches-beta-test/|title= Communote launches Beta-test |publisher=Communote}} Like [[Twitter]], it allows users to post updates of their activities. Unlike Twitter, Communote is a "professional microblogging service"{{cite web|accessdate=2009-06-17|url=http://blogs.zdnet.com/Hinchcliffe/?p=414&page=3|title=Twitter on your intranet: 17 microblogging tools for business |publisher=ZDNet}} focusing on the internal usage in companies. It is available as in-house installation and as online service. == Features == Communote differs from standard enterprise [[microblogging]] tools due to its special microsharing approach. This means that storage and retrieval of postings is well supported, especially by tagging and extensive filtering features.{{cite web|accessdate=2009-06-17|url=http://microblogging.com/2009/03/17/interview-with-communardo-ceo-dirk-rohrborn/|title=Interview with Communoteâ€™s CEO |publisher=microblogging.com}} Further features are as follows: * [[Tag_(metadata) | Tagging (including hashtags)]] * [[LDAP]] integration * File attachments * Sending and receiving postings via [[instant message|IM]] and [[Email]] * Mobile client * Design customization * Extensive filtering and RSS feeds on every possible filter combination * [[Portlets]] * [[JSON]] [[API]] * [[Transport Layer Security|SSL]] ==Technology== In difference to common web 2.0 applications including other [[microblogging]] tools Communote is built on [[Java EE]] technology following a classical [[software engineering]] approach. == See also == * [[Jaiku]] * [[OpenMicroBlogging]] * [[Plurk]] * [[Pownce]] * [[Present.ly]] * [[Twitter]] * [[Yammer]] == References == {{reflist|1}} == External links == * [http://www.communote.com Communote] * [http://pistachioconsulting.com/microsharing/enterprise-microsharing-tools-comparison-ebook/ Enterprise Microsharing Tools Comparison] [[Category:Twitter]] [[Category:Social network services]] [[Category:Web 2.0]] [[Category:Blog hosting services]] [[Category:Internet properties established in 2008]]
