{{Unreferenced|date=July 2009}} '''Local Security Authority Subsystem Service''' (LSASS), is a [[Process (computing)|process]] in [[Microsoft Windows]] [[operating system]]s that is responsible for enforcing the [[security policy]] on the system. It verifies users logging on to a Windows computer or server, handles password changes, and creates [[access token]]s. It also writes to the [[Windows Security Log]].
