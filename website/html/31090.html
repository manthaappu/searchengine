{{Use ymd dates|date=September 2010}} {{Infobox Software | name = RabbitMQ | logo = [[Image:RabbitMQLogo.png|200px]] | screenshot = | caption = | developer = [[SpringSource]] (division of VMware) | latest release version = 2.2.0 | latest release date = {{release date|2010|11|30}} | latest preview version = | latest preview date = | operating system = [[Cross-platform]] | programming language = [[Erlang (programming language)|Erlang]] | genre = [[Advanced Message Queuing Protocol|AMQP]], [[message-oriented middleware]] | license = [[Mozilla Public License]] | website = [http://www.rabbitmq.com/ www.rabbitmq.com] }} In computing, '''RabbitMQ''' is an open source [[message broker]] software ([[message-oriented middleware]]), using the standard [[Advanced Message Queuing Protocol]] (AMQP). The RabbitMQ server is written in [[Erlang (programming language)|Erlang]] and is built on the [[Open Telecom Platform]] framework for clustering and failover. Rabbit Technologies Ltd., acquired in 2010 April by [[SpringSource]], develops and provides support for RabbitMQ.{{cite press release |date=2010-04-13 |title=Rabbit Technologies announce acquisition by SpringSource |url=http://www.rabbitmq.com/news.html }} The source code is released under the [[Mozilla Public License]]. The RabbitMQ project consists of several parts: * The RabbitMQ exchange server itself * Gateways for [[HTTP]], [[Extensible Messaging and Presence Protocol|XMPP]] and [[Streaming Text Orientated Messaging Protocol|STOMP]] protocols * AMQP client libraries for [[Java (programming language)|Java]] and [[.NET Framework]].
