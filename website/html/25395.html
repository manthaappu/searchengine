{{Multiple issues|orphan =December 2010|wikify =September 2009|refimprove =September 2009|copy edit=September 2009}} '''Multiscale Electrophysiology Format''' (MEF) was developed to handle the large amounts of data produced by large-scale electrophysiology in human and animal subjects. MEF can store any time series data up to 24 bits in length, and employs lossless range encoded difference compression. Subject identifying information in the file header can be encrypted using 128-bit AES encryption in order to comply with HIPAA requirements for patient privacy when transmitting data across an open network. Compressed data is stored in independent blocks to allow direct access to compressed data, facilitate parallel processing, and limit the effects of potential damage to file. Data fidelity is ensured by 32-bit cyclic redundancy check in each compressed data block using the Koopman polynomial (0xEB31D82E), which has a Hamming distance of 4 up to 114 kbits.
