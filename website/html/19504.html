The JBoss Enterprise SOA Platform is part of the JBoss Enterprise Middleware portfolio of software.{{cite web|title=JBoss Enterprise Middleware|url=http://www.redhat.com/jboss/|publisher=[[Red Hat]]}} The JBoss Enterprise SOA Platform enables enterprises to integrate services, handle business events, and automate business processes, linking IT resources, data, services and applications.{{cite web|title=JBoss Enterprise SOA Platform|url=http://www.jboss.com/pdf/SOA_infosheet.pdf|publisher=[[Red Hat]]}} Because it is [[Java (programming language)|Java]]-based, the JBoss application server operates cross-platform: usable on any operating system that supports [[Java (programming language)|Java]]. The JBoss SOA Platform was developed by [[JBoss (company)|JBoss]], now a division of [[Red Hat]]. ==Product features and components== * [[Business rules engine]] * [[JBPM|JBoss jBPM]] for workflow for Java developers * JBoss Enterprise Service Bus (ESB) * Event management and [[complex event processing]] (CEP) * [[Event-driven architecture]] (EDA) * [[Data integration]] * [[Service-oriented architecture|Service Oriented Architecture]] (SOA) services * [[JBoss Messaging]] * JBoss Enterprise Application Platform (EAP) * [[Java Platform, Enterprise Edition|Java Enterprise Edition]] (JEE) services * [[Computer clustering]] * [[Java EE Connector Architecture|Java Connector Architecture]] (JCA) ==Enterprise Service Bus (ESB)== The '''JBoss Enterprise Service Bus''' (ESB) (or '''JBossESB''') software is part of the JBoss Enterprise SOA Platform. The software is [[Enterprise application integration|Enterprise Application Integration]] (EAI) or business integration software. In general, [[enterprise service bus]] software is used to map the abstract SOA concepts onto a concrete implementation. The software is [[middleware]] used to connect systems together, especially non-interoperable systems.
