'''MacroML''' is an experimental [[programming language]] based on the [[ML programming language|ML]] programming language family that seeks to reconcile ML's [[static typing]] systems, and the types of [[Macro (computer science)|macro]] systems more commonly found in [[dynamically typed]] languages like [[Scheme (programming language)|Scheme]]; this reconciliation is difficult as macro transformations are typically [[Turing-complete]] and so can break the [[type safety]] guarantees static typing is supposed to provide. == External links == Some papers related to MacroML include: *[http://citeseer.ist.psu.edu/440438.html "Macros as Multi-Stage Computations: Type-Safe, Generative, Binding Macros in MacroML"] ([[citeseer]]) *[http://citeseer.ist.psu.edu/643840.html Staged Notational Definitions] (citeseer) {{DEFAULTSORT:Macroml}} [[Category:Experimental programming languages]] [[Category:ML programming language family]]
