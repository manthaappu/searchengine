'''PortMusic''' is a set of [[computer library|computer libraries]] for dealing with [[sound]] and [[MIDI]]. Currently the project has two main libraries: [[PortAudio]], for [[digital audio]] input and output, and [[PortMidi]], a library for [[MIDI]] input and output. A library for dealing with different audio [[file format]]s is planned, although another library, [[libsndfile]], already exists and is licensed under the liberal [[GNU Lesser General Public License]]. ==External links== * [http://www.cs.cmu.edu/~music/portmusic/ PortMusic website] [[Category:Computer libraries]] [[Category:Free audio software]] [[Category:Audio libraries]] [[pt: PortMusic]]
