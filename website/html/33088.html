{{Infobox OS | name = Sabily GNU/Linux | logo = [[Image:SabilyLogo-128.png|128px|Sabily logo]] | screenshot = [[Image:Sabily-menu.png|250px]] | caption = Sabily | website = [http://www.sabily.org www.sabily.org] | family = [[Unix-like]] | source_model = [[Open source]] | latest_release_version = 10.10 (Al-Quds) | latest_release_date = {{Start date and age|2010|11|03}} | language = Multilingual (more than 55){{Citation needed|date=December 2010}} | kernel_type = [[Monolithic kernel|Monolithic]] ([[Linux kernel|Linux]]) | ui = [[GNOME]] | license = [[GNU GPL]] and others | updatemodel = [[Advanced Packaging Tool|APT]] (front-ends available) | package_manager = [[dpkg]] | supported_platforms = [[x86]][http://www.sabily.org/website/en/sabily/downloads.html Sabily downloads] }} '''Sabily''' ({{lang-ar|سبيلي}}, {{IPA-ar|sæˈbiːliː|IPA}}, ''My Way'') is a free, open source operating system designed by and for [[Muslims]]. It is based on the popular{{Citation needed|date=December 2010}} [[Ubuntu (operating system)|Ubuntu]] [[Linux distribution]] which is a Free and Open Source ([[FOSS]]) operating system and has pre-installed packages for the [[Arabic]] language. Sabily includes a set of packages that customizes Ubuntu. Ubuntu is easy to use for a novice user{{Citation needed|reliable|date=December 2010}},[https://help.ubuntu.com/9.10/switching/why-easy.html Ubuntu is easy to use] with the most-used Office, Internet, Graphics and Video applications already installed{{Citation needed|date=December 2010}}. With Sabily, [[Muslim]] users have “out of the box” [[Islamic]] software and tools installed as well. Sabily includes [[Islamic]] software (prayer times tool, [[Quran]] study tool, [[Hijri]] calendar etc.) and has a custom design. The full version of Sabily comes with out of the box educational software, [[Codec|codecs]] for the most used media formats{{Citation needed|codecs for the most used media formats|date=December 2010}}.[http://www.sabily.org/website/en/sabily/content.html Sabily content] == Versions available == {{Inappropriate tone|date=June 2009}} Sabily is available as a Live DVD, which can be booted on the host computer without installation. Currently there are three versions existing: * Small version (967.96 MB): contains the main Sabily packages, including artwork and Islamic applications and Arabic support * Full version (1.55 GB): contains the same as the Small version plus multimedia, educational and miscellaneous packages (see the full list [http://www.sabily.org/website/index.php?option=com_content&view=article&id=2&Itemid=9&lang=en here]) * Full version with recitations (3.3 GB): same as Full version plus Quran recitations provided by http://www.versebyversequran.com: Muhammad Siddeeq al-Minshawi, Huzify, Sa’ad al-Ghamadi and Mishary Rashed Alafasy.
