{{Infobox software | name = | logo = | screenshot = [[Image:Sputnik web browser screenshot.png|256px|Sputnik displaying Wikipedia front page on 14 July 2008]] | caption = Sputnik displaying [[Wikipedia]] front page on 2008-06-14 | collapsible = | author = | developer = | released = | discontinued = | latest release version = | latest release date = | latest preview version = | latest preview date = | frequently updated = | programming language = | operating system = [[MorphOS]] | platform = | size = | language = | status = | genre = | license = [[LGPL]] | website = http://khtml.ppa.pl/ }} '''Sputnik''' is a web-browser for [[MorphOS]] developed by Marcin Kwiatkowski. It is a port of the [[KHTML]] browser, which is based on [[S60 platform|S60]] [[WebKit|WebCore]]. Sputnik was first released to the general public on November 11, 2006.{{Cite web| url=http://www.morphzone.org/modules/news/article.php?storyid=1299 | title=Announcements : SPUTNIK BROWSER finally available! | author=JoBBo | date=November 11, 2006 | work=MorphZone | accessdate=2007-03-19 }} The second public release occurred on March 10, 2007.{{Cite web| url=http://amigaweb.net/index.php?function=view_news&id=407 | title=New Sputnik Beta released! | author=Tommy Strand | date=March 10, 2007 | work=AmigaWeb.Net | accessdate=2007-03-19 }}{{Cite web| url=http://www.morphzone.org/modules/news/article.php?storyid=1350 | title=Announcements : Sputnik Web Browser - Second beta released! | author=JoBBo | date=March 10, 2007 | work=MorphZone | accessdate=2007-03-19 }} The third beta was released on August 9, 2007. Sputnik is the first browser for MorphOS capable of modern standards such as [[CSS]].{{Cite web| url=http://www.totalamiga.org/issue25.html | title=Interview with Marcin Kwiatkowski | author=Robert Williams | date=Winter 2006/2007 | work=Total Amiga Magazine, issue 25, page 18 | accessdate=2007-03-19 }} The authors of MorphOS arranged to include Sputnik in their 2.0 version (M2), released on 30 June 2008. The Sputnik in M2 is dated 2 June 2008, according to its "About" window. The other web-browsers available for MorphOS include [[AWeb]], [[IBrowse]], and [[Voyager (web browser)]], [[NetSurf]], [[WebKit#Ports|OWB]] (Origyn Web Browser). In June 2009, the developer said that Sputnik development would continue, but "it'll be a different browser." {{Cite web| url= http://groups.google.com/group/comp.sys.amiga.morphos/browse_thread/thread/a9f531f58b21528f | title=Usenet post on M.
