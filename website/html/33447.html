'''ScriptBasic''' is a [[scripting language]] variant of [[BASIC programming language|BASIC]]. The source of the [[interpreter]] is available as a [[C (programming language)| C program]] under the [[GNU Lesser General Public License|LGPL]] license. ScriptBasic generates intermediary code which is then interpreted by a runtime environment. ScriptBasic is available for [[Microsoft Windows|Windows]], [[Unix]] and Mac OS X and may be embedded in other programs as well. It can create standalone [[executable]] files. A runtime library is linked into the executable. It is available in precompiled binaries (setup.exe under Windows and uninstall also supported), [[dpkg]] and [[RPM Package Manager|rpm]] for [[Linux]] and in source code form. The language, the interpreter is fully documented in the Users' Guide available in text, [[HTML]], [[CHM]], [[TeX]], [[texi]] and [[Portable Document Format|PDF]] formats. ScriptBasic has been developed since 1999 and has reached a fairly matured state in terms of functions and stability. The precompiled version available for Windows and Linux includes a [[command line]] version and a standalone [[web server]]. This BASIC can be the choice for developers, who seek a BASIC variant that runs on UNIX as well as under Windows and Mac OS X (Intel). The Basic is embeddable with an option to compile your applications to a small footprint executable. ScriptBasic has an open interface for module developers. There are several external modules developed by the developer of ScriptBasic as well as by other developers. These include data base connection handling for various [[RDBMS|database]] systems ([[MySQL]], [[PostgreSQL]], [[ODBC]], [[Berkeley DB]] and others), binding to the library [[CURL]], [[Portable Network Graphics|PNG]] graphics, [[GTK+]] [[graphical user interface]], [[Internet socket|sockets]], [[regular expression]]s, [[thread (computer science)|thread]] support, [[data compression]] and [[Common Gateway Interface|CGI]]. ScriptBasic also has an open interface for [[preprocessor]] developers. These are modules that may act not only during run-time but also compile time, thus making it possible to alter the language. Currently there is a single preprocessor that delivers [[debugger]] functionality. This lets the BASIC programmer to run the BASIC program line by line, examine variable contents, set break points and all the usual debugging features. This debugger supports not only the command line version but also the web server implementation allowing full interactive debugging of CGI applications in BASIC. The architecture of the interpreter internally is [[object oriented]] and provides a clean and well documented interface to embed the interpreter into any application written in [[C (programming language)|C]] or [[C++]]. The whole source code is extensively documented and commented, which is an outstanding feature compared to other embeddable script language implementations. Slides in HTML format with English narration in [[RealAudio]] format are also available to get a jump start learning the architecture and module, preprocessor and embedding developments. ScriptBasic is supported by a [http://www.scriptbasic.org/forum/ forum]. ==Features== * Open source LGPL project * Multi-platform (Windows, Linux, Unix, Mac OS X, ...) * Fast multi-threaded ScriptBasic HTTP application server * In memory session handling * Extension module API (CGI, MySQL, ODBC, Berkeley DB, GTK, Curl, ...) * HTML template file support * Command line interpreter for utility applications and standard CGI programming * Create standalone executables without compiling (This bundles the interpreter and code together into an executable file) * Name space support * Relaxed / auto variable typing (forget DIM and think undef) * TYPE, [[Is functions|ISARRAY, ISSTRING, ISINTEGER, ISREAL, ISNUMERIC, ISDEF, ISUNDEF, ISEMPTY]] * Associative arrays (reference elements by name - MySQL & ODBC modules assign columns to named arrays) * Standard branching syntax ([[GOTO]], [[GOSUB]]/[[Return statement|RETURN]], [[For loop|FOR/NEXT]], DO/WHILE/UNTIL, REPEAT/UNTIL, CALL) * Multi-function IO directives ([LINE] INPUT, PRINT, OPEN - files, ports, standard IO) * Time/Date formatting and math * Extended string functions (REPLACE, SPLIT, SPLITA) * GTK / Glade support for desktop application development * Easy to learn and use without prior programming knowledge ==External links== *http://www.scriptbasic.org/ — ScriptBasic open source project site.
