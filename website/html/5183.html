{{unreferenced|date=December 2009}} {{Infobox Software | name = CPU Speed Pro | screenshot = [[File:CPU-Speed-Pro.png|250px]] | caption = CPU Speed Pro 3.0.3.6 under [[Windows 7]] | developer = [http://www.cpuspeedpro.com/ Wizard Software] | latest_release_version = 3.0.3.6 | latest_release_date = {{release_date|2009|12|31}} | operating_system = [[Microsoft Windows]] | genre = [[System profiler]] | license = [[Freeware]] | website = [http://www.cpuspeedpro.com/ www.cpuspeedpro.com] }} '''CPU Speed Pro''' is a [[freeware]] [[system profiler]] [[application software|application]] for [[Microsoft Windows]] that detects the type of [[central processing unit]] (CPU) located on a [[motherboard]] and tests the speed of a motherboard's processor in four automated steps. The ability to accurately check a computer system's CPU speed is an important feature for [[overclockers]] who wish to overclock their motherboard and CPU. Processor speed results are displayed in a pop-up window, where users can submit their CPU benchmark results to an online database and compare their processor speed with other users around the world. CPU speed results are stored in an online database and categorized by CPU manufacturers ([[Intel]], [[AMD]]), CPU types and countries.
