{{About|Noop, the programming language|the processor instruction (usually written "NOP")|NOP}} {{Infobox programming language | name = Noop | typing = [[static typing|static]] | year = 2009 | license = [[Apache License]] 2.0 | website = http://code.google.com/p/noop }} '''Noop''' ({{IPA-en|ˈnoʊ.ɒp|pron}}, like [[NOP]]) is a new project from [[Google]] aiming to develop a new programming language. Noop attempts to blend the best features of "old" and "new" languages, while syntactically encouraging industry best-practices.
