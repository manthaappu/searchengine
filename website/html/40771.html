{{Unreferenced stub|auto=yes|date=December 2009}} {{Orphan|date=December 2009}} '''xconfig''' is short for the 'xconfig' target for the [[Linux Kernel|Linux]] [[Makefile]]. It is a graphical Linux compilation utility, which uses [[Qt (toolkit)|Qt]]. The xconfig utility is invoked by running 'make xconfig' in the base Linux source directory. Qt has only been used in Kernel 2.6. Kernel 2.4 uses a [[Tcl]]/[[Tk (computing)|Tk]] configuration interface. A [[GTK+]] interface is also available for Kernel 2.6.
