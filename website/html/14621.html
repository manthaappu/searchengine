It is characterized by strict separation of data storage, processing and user interface development. GNUstepWeb is based on [[GNUstep]]. Its goals are to be compatible to [[WebObjects]] version 4.5.x with some extensions. [[Apple Inc.|Apple]]'s WebObjects was ported to [[Java (programming language) | Java]] with version 5.
