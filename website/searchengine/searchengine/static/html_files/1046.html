{{Orphan|date=January 2009}} {{Unreferenced|date=March 2009}} {{ infobox software | name = Admon | screenshot = [[Image:Zabbix.png|200px]] | caption = Admon Original version Web interface | developer = Admon | latest_release_version = 2.0 | latest_release_date = Sep 2008 | operating_system = [[Linux]] | genre = [[Network monitoring]] | license = [[GNU General Public License]] | website = http://admon.org/ }} __NOTOC__ '''Admon''' is a [[Zabbix]]-based [[open-source]] [[computer system]] and [[network monitoring]] [[application software]]. It watches [[node (networking)|host]]s and services specified by users, alerting them via e-mail or short-message service when issues arise. Created under [[Zabbix]] v1.1.1 beta, Admon was modified to run under [[Linux]] only, but it also runs on other [[Unix]] variants. Admon is [[free software]] licensed under a [[GNU General Public License]], version 2, as published by the [[Free Software Foundation]]. Compared to Zabbix, most of the original functions have been shrunk, including supports for windows, multiple databases, [[simple network management protocol]], and auto-discovery; and only some core functions are reserved. Its alerting subsystem has been rewritten for easier use in large-scale networks.
