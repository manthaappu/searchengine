{{orphan|date=November 2009}} An '''Active Database''' is a [[database]] that includes an event driven architecture (often in the form of [[Event Condition Action|ECA rules]]) which can respond to conditions both inside and outside the database. Possible uses include security monitoring, alerting, statistics gathering and authorization.
